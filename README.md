# Condensador

Este repositorio tendra el fin de explicar detalladamente como funcionan los condensadores en corriente continua y alterna, con el fin de comprender todo su funcionamiento, entre esta investigación se encuentran diversos calculos aplicados a la aplicación y naturalidad del condensador.

Un condensador es un aparato electronico que tiene la capacidad de almacenar carga en sus placa, esta construido por dos placa paralelas con la misma superficie separadas entre si mediante una determinada distancia, este es un condensador ideal y el mas representado para comprender su funcionamiento.
